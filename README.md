# ToDoアプリ

HTML、CSS、JavaScriptで作成したシンプルなToDoアプリです。

## デモ

https://strangehumen.github.io/todo/

## 制作背景

日々のタスクを手軽に管理できるアプリが欲しかったため制作しました。

シンプルな操作性とスマートフォンでの使いやすさを重視しています。

## 主な機能

- タスク追加
- タスク完了チェック
- タスク削除
- タスク編集
- 完了済みタスクの一括削除
- フィルター機能（全て・未完了・完了済み）
- 未完了タスク数の表示
- localStorageによるデータ保存
- ダークモード
- レスポンシブデザイン

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 特徴

- ブラウザを閉じてもデータが保持される
- スマートフォンでも快適に利用可能
- シンプルで直感的なUI
